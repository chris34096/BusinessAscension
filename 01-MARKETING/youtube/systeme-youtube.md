## 1. RÔLE DE YOUTUBE DANS L'ÉCOSYSTÈME

YouTube = **le réseau de la profondeur et de la pré-vente**. L'actif long terme : une vidéo travaille pour toi pendant des années.

- **Objectif** : autorité maximale + pré-vente profonde → le prospect arrive à l'Audit offert / à la Porte 1 déjà convaincu.
- **Ce que YouTube fait mieux** : installer une croyance en profondeur (8-15 min) que LinkedIn/IG ne font qu'effleurer. **C'est le terrain naturel de YAP** : une vidéo = une croyance défendue à fond pendant 10 minutes.
- **C'est le réservoir** : chaque vidéo se découpe en dizaines de Reels, Shorts, posts, carrousels.

---

## 2. LE PRINCIPE — YAP en format long (le plus exigeant)

> Chaque vidéo enseigne un **mécanisme complet** — assez pour un vrai résultat seul. La générosité = la preuve. "Il donne ça gratuitement, imagine l'accompagnement."

YouTube est la maison de YAP. La vidéo longue suit la **formule YAP en 5 temps** (`yap-framework.md` §1) déployée sur 8-15 min :
1. Pose la croyance (le hook + le titre Want).
2. Explique le pourquoi.
3. Donne la preuve (cas, mécanisme, démonstration écran).
4. Élargis l'idée (le pourquoi du pourquoi — la profondeur qui crée l'autorité).
5. Pose l'avion (takeaway mémorable + 3 CTA Easy Next Yes).

Tom Pearson : chaque vidéo installe **une croyance précise** (les 8 → `vsl-script.md`). Taki Moore : on **enseigne ET on vend** dans la même vidéo, jamais séquentiellement.

## 3. MÉCANIQUE (Want vs Need — Taki Moore)

- Le titre/hook vend ce que la cible **VEUT** (CA prévisible, leads, vendre par message).
- Le contenu livre ce dont elle a **BESOIN** (le mécanisme, l'identité, le système).

| Étage | Ce qui le crée sur YouTube |
|---|---|
| **+ Abonnés** | Titre Want + miniature + 30 premières s qui tiennent la promesse |
| **+ Autorité** | Profondeur YAP du mécanisme, de bout en bout |
| **+ Confiance** | Cas clients + ton parcours (12K→0) + "je te donne tout" |
| **+ Clients** | 3 CTA selon température (froid/tiède/chaud) → Audit offert / Porte 1 |

## 4. STRUCTURE D'UNE VIDÉO LONGUE (gabarit YAP)

```
[0:00-0:30] HOOK — pas d'intro. Croyance + friction. Want first. (YAP temps 1)
[0:30-2:00] QUI / POURQUOI M'ÉCOUTER — crédibilité rapide + enjeu
[2:00-4:00] LE VRAI PROBLÈME — diagnostic, casser la fausse croyance (YAP temps 2)
[4:00-10:00] LE MÉCANISME — valeur pure, étape par étape + preuve (YAP temps 3-4, Need)
[10:00-12:00] LA PREUVE + LIGNE DANS LE SABLE — cas / démo, ancien vs nouveau chemin
[12:00-14:00] LAND THE PLANE + 3 CTA — takeaway + Easy Next Yes (YAP temps 5)
```
Détail : `structure-video-youtube.md` · `vsl-script.md`.

## 6. LA CADENCE HEBDOMADAIRE (proposée)

> YouTube = qualité avant volume. Réaliste pour un fondateur solo qui fait déjà 14 reels IG/semaine. Le long-form ne se batche pas comme un reel.

| Pièce | Fréquence | Source |
|---|---|---|
| **1 vidéo longue (8-15 min)** | **1 / semaine** — le flagship | 1 croyance, gabarit YAP §4 |
| **3-4 Shorts** | **3-4 / semaine** | Reels IG / Daily YAP recyclés (vertical) |
| **1 podcast / format long bonus** | **1 / 2 semaines** (optionnel) | conversation, clip YAP §4 format 9 |

**Volume hebdo cible :** 1 long-form + 3-4 Shorts.

**Workflow :**
1. **Lundi** — choisir l'angle (Want fort) + définir LA croyance à installer + le style YAP.
2. Écrire le script complet (gabarit §4) via YAP — souvent depuis le Daily YAP de la semaine, approfondi.
3. **Tourner** (face cam, voix Chris — assertions, phrases courtes). Titre + miniature Want explicite (`analyse-concurrents-youtube.md`).
4. **Atomiser** : la vidéo → 4-6 Reels + 3-4 Shorts + 2-3 posts LinkedIn + 1-2 carrousels.
5. Description : lien **Porte 1 (Deviens l'Entrepreneur™)** + **Audit offert**.

---

## 7. LE CHEMIN DE CONVERSION (3 CTA — Easy Next Yes)

- **Froid** (découvre la chaîne) → "Regarde cette autre vidéo : [lien]" (rétention).
- **Tiède** (te suit) → "Commente AUDIT / récupère le [lead magnet]".
- **Chaud** (prêt) → "Lien en description : Audit offert / Construis la Marque™."

Coach Mike : chaque vidéo a un chemin clair vers une action. Jamais "merci d'avoir regardé" sans CTA.
