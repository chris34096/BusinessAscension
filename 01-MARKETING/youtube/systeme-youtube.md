## 1. RÔLE DE YOUTUBE DANS L'ÉCOSYSTÈME

YouTube = **le réseau de la profondeur et de la pré-vente**. C'est l'actif long terme : une vidéo travaille pour toi pendant des années.

- **Objectif** : autorité maximale + pré-vente profonde → le prospect arrive à l'Audit offert / au low ticket déjà convaincu.
- **Ce que YouTube fait mieux** : installer des croyances en profondeur (10-15 min) que LinkedIn/IG ne peuvent qu'effleurer.
- **C'est le réservoir** : chaque vidéo se découpe en dizaines de Reels, posts, carrousels.

---

## 2. LE PRINCIPE VALUE-FIRST (le plus exigeant des 3 réseaux)

> Chaque vidéo enseigne un **mécanisme complet** — assez pour que le spectateur obtienne un vrai résultat seul. La générosité = la preuve. Si "il donne ça gratuitement, imagine l'accompagnement."

Tom Pearson : chaque vidéo installe une **croyance précise** (les 8 croyances → `vsl-script.md`). Taki Moore : on **enseigne ET on vend** dans la même vidéo, jamais séquentiellement.

---

## 3. MÉCANIQUE DE RÉSULTATS (Want vs Need — Taki Moore)

- Le titre/hook vend ce que la cible **VEUT** (CA prévisible, leads, vendre par message).
- Le contenu livre ce dont elle a **BESOIN** (le mécanisme, l'identité, le système).

| Étage | Ce qui le crée sur YouTube |
|---|---|
| **+ Abonnés** | Titre Want + miniature + premières 30s qui tiennent la promesse |
| **+ Autorité** | Profondeur du mécanisme enseigné de bout en bout |
| **+ Confiance** | Cas clients + ton parcours + "je te donne tout" |
| **+ Clients** | 3 CTA selon température (froid/tiède/chaud) → low ticket / Audit offert |

---

## 4. STRUCTURE D'UNE VIDÉO (le gabarit)

```
[0:00-0:30] HOOK — pas d'intro. La promesse + la friction. Want first.
[0:30-2:00] QUI / POURQUOI M'ÉCOUTER — crédibilité rapide + enjeu
[2:00-4:00] LE VRAI PROBLÈME — diagnostic, casser la fausse croyance
[4:00-10:00] LE MÉCANISME — la valeur pure, enseignée étape par étape (Need)
[10:00-12:00] LA PREUVE — cas / démonstration + ligne dans le sable (ancien/nouveau chemin)
[12:00-14:00] PRODUCT PLACEMENT + EASY NEXT YES — 3 CTA selon température
```
Détail complet : `structure-video-youtube.md` · `vsl-script.md`.

---

## 5. LES 4 TYPES DE VIDÉOS

### A — LE MÉCANISME (autorité fondatrice)
Enseigne une des 8 étapes ou un pilier (vendre par message, machine à leads, l'imprévisibilité...). C'est le cœur. *(cf. VA01-VA06)*

### B — LE "COMMENT" TACTIQUE (Want fort)
"Comment passer à 10K€/mois réguliers" / "Comment vendre par message". Promesse directe + méthode complète.

### C — L'ANALYSE / TEARDOWN (expertise)
Décortiquer une erreur, un cas, une stratégie concurrente. Montre comment tu penses.

### D — LA STORY / VISION (confiance + marque)
Ton parcours (12K→0), ta vision, le build in public. Crée l'attachement à la marque.

## 6. LE SYSTÈME DE PRODUCTION (l'OS répétable)

**Rythme réaliste :** 1-2 vraies vidéos longues/semaine (la qualité prime). Les 30 angles du `calendrier-omnipresence-30j.md` alimentent le pipeline.

**Workflow :**
1. Choisir l'angle (Want fort) + définir LA croyance à installer.
2. Écrire le script complet (skill [[02-prisme]] ou agent ContentEngine) selon le gabarit §4.
3. Tourner (face cam, direct, voix Chris — assertions, phrases courtes).
4. Titre + miniature : Want explicite (cf. `analyse-concurrents-youtube.md` pour les patterns).
5. **Découper la vidéo** en 4-6 Reels + 2-3 posts LinkedIn + 1-2 carrousels (omniprésence).
6. Description : lien **low ticket 7-27€** + **Audit offert**.

---

## 7. LE CHEMIN DE CONVERSION (3 CTA — Easy Next Yes)

- **Froid** (découvre la chaîne) → "Regarde cette autre vidéo : [lien]" (rétention).
- **Tiède** (te suit) → "Commente AUDIT / télécharge le [lead magnet ou low ticket]".
- **Chaud** (prêt) → "Lien en description : Audit offert / Construis la Marque™."

Coach Mike : chaque vidéo a un chemin clair vers une action. Jamais "merci d'avoir regardé" sans CTA.

---

## 8. INTERCONNEXION ÉCOSYSTÈME (YouTube = le réservoir)

```
1 vidéo YouTube (10-15 min)
   ├── 4-6 Reels Instagram (moments forts)
   ├── 2-3 posts LinkedIn (frameworks extraits)
   ├── 1-2 carrousels (la méthode visualisée)
   └── 1 newsletter / email
```
→ YouTube produit la matière première. [[02-prisme]] fait l'atomisation. C'est le pilier de l'omniprésence.
