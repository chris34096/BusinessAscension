## 1. RÔLE DE LINKEDIN DANS L'ÉCOSYSTÈME

LinkedIn = **le réseau de l'autorité écrite**. C'est là que l'ICP 2 (entrepreneurs établis) décide s'il te fait confiance.

- **Objectif** : autorité perçue → confiance → leads entrants présoldés → clients par message (Porte 2 + low ticket).
- **Ce que LinkedIn fait mieux que les autres** : prouver la profondeur de ta pensée. Un post écrit dense = preuve d'expertise instantanée.
- **Interconnexion** : un post LinkedIn → angle de Reel (IG) → segment de vidéo (YouTube). Voir §8.

## 2. LE PRINCIPE VALUE-FIRST (non négociable)

> Chaque post doit donner un **résultat utilisable immédiatement** — au point que le lecteur se dise "il vient de me donner gratuitement ce que d'autres vendent en formation".

Règle : si quelqu'un applique TON post et n'obtient rien de concret → le post a échoué. Chaque post enseigne **un framework complet, pas un teaser.**

**Test de qualité avant publication :**
- [ ] Le lecteur peut-il agir dès aujourd'hui avec ça ?
- [ ] Y a-t-il un mécanisme (le "comment"), pas juste un constat ?
- [ ] Est-ce que ça installe une croyance qui mène à BA™ ?
- [ ] Est-ce dans la voix de Chris (direct, oral, tutoiement, VOC) ?

## 4. LES 5 FORMATS QUI GAGNENT SUR LINKEDIN

### Format A — LE FRAMEWORK (autorité maximale)
Enseigne un système complet, étape par étape. Le post le plus sauvegardé.
> Structure : Hook (croyance) → le problème → le framework en 3-5 étapes numérotées → exemple → CTA.

### Format B — LA STORY (confiance maximale)
Ton parcours, vulnérable, avec une leçon business. Le post le plus commenté.
> Structure : Hook (moment précis) → la galère → le déclic → ce que ça t'a appris → CTA.

### Format C — L'OPINION TRANCHÉE (reach + filtre)
Une prise de position contrarian qui divise. Attire les bons, repousse les mauvais.
> Structure : Affirmation choc → pourquoi le consensus a tort → ta vérité + preuve → CTA.

### Format D — LE TEARDOWN / DIAGNOSTIC (expertise démontrée)
Tu décortiques une erreur ou un cas en direct. Montre comment tu penses.
> Structure : "Voici pourquoi [X] échoue" → diagnostic en couches → le fix → CTA.

### Format E — LA LISTE DE VALEUR (sauvegardes)
"X erreurs / signes / leviers" — dense, scannable, actionnable.
> Structure : Hook chiffré → liste (chaque item = problème + fix) → le plus important → CTA.

## 5. LE SYSTÈME DE PRODUCTION (l'OS répétable)

**Rythme cible :** 1 post/jour minimum (2/jour en phase de croissance — 1 autorité le matin, 1 story/engagement le soir).

**Workflow hebdomadaire (le 5-2 LinkedIn) :**
1. **Lundi** — choisir 5 idées (1 par format A-E) depuis `calendrier-omnipresence-30j.md` ou un angle du jour.
2. **Production batch** — écrire les 5 en un bloc (skill [[02-prisme]] ou agent ContentEngine).
3. **2 hooks A/B** par post → garder le plus fort.
4. **Publier + commenter** : 10-20 commentaires stratégiques/jour sous les posts de l'ICP 2 et des concurrents (Musy, Baer, Clouet).
5. **Logger** vues/commentaires/DM → doubler sur les formats qui sortent (outliers).

**Structure d'un post (mise en forme LinkedIn) :**
```
[HOOK — 1 ligne, < 12 mots, frappe directe]

[1 ligne de transition — pose la promesse]

[CORPS — lignes courtes, 1 idée par ligne, très aéré]

[INSIGHT PIVOT — la phrase à retenir, isolée]

[CTA — 1 ligne]

(0-3 hashtags précis, jamais génériques)
```

## 8. INTERCONNEXION ÉCOSYSTÈME

Un post LinkedIn ne meurt pas sur LinkedIn :
- **Framework LinkedIn** → script **Reel** IG (format "X erreurs/étapes") → segment **YouTube**.
- **Story LinkedIn** → Reel story IG → intro de vidéo YouTube.
- Le post le plus performant de la semaine = sujet de la prochaine vidéo YouTube longue.

→ 1 idée nourrit les 3 réseaux. C'est l'omniprésence (skill [[02-prisme]] automatise le repurposing).
