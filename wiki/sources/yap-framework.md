# YAP Framework + Guide Viralité 2026
Deux ressources externes ingérées et fusionnées dans la référence canonique `01-MARKETING/_fondations/yap-framework.md` (v1.1). Moteur d'idéation + de script pour tout le contenu BA™ (Instagram · YouTube · LinkedIn).

🔗 REF EXTERNE — sources : *The Online Coach YAP Guide* (formule + 5 styles) et *Guide Viralité 2026* (tunnel + exécution). Adaptées à la voix Chris, VOC v2, belief-first.

## Ce que la source apporte
**YAP = approfondir une idée, pas courir après le hook.** 80% sur la profondeur de la croyance ; le hook devient évident quand l'idée est profonde. Chaque pièce installe UNE croyance (Tom Pearson, cf. [[voix-chris]]).

- **Formule YAP (5 temps)** : Pose la croyance → Explique le pourquoi → Donne la preuve → Élargis l'idée → Pose l'avion.
- **5 styles** : Contrarian (reach+filtre) · Story (confiance) · Breakdown (expertise) · Rant (autorité) · Rabbit Hole (profondeur).
- **Daily YAP** : 1 croyance + 5 questions = 3-10 min de matière brute → découpée en reels/carrousel/post/segment.
- **Short-Form Script Builder** : process 3 étapes (sujet → structure Hook/Body/CTA → script ligne par ligne) — existe en Projet Claude à copier (§6).
- **Tunnel + viralité** (Guide 2026) → page dédiée [[tunnel-tofu-mofu-bofu]] : répartition 4/3/3, hook 3s, rétention, discipline quotidienne, analyse des flops, "ils achètent toi".

## Manifestations BA™
- Référence canonique consommée par les 3 systèmes réseaux (`systeme-instagram/youtube/linkedin.md`) et injectée brute dans l'agent Contenu (`run-agent-contenu.ps1`).
- Mapping styles → étages tunnel → CTA documenté dans [[tunnel-tofu-mofu-bofu]].
- Voix : tutoiement, oral, mots bannis (liberté financière, revenus passifs, système clé en main, "dents de scie"…), pas de tirets cadratins dans les scripts publiés.

## Observations
- YAP (profondeur) × Tunnel 4/3/3 (conversion) = les deux axes orthogonaux d'une pièce de contenu : *quoi dire à fond* × *pour quel rôle*.
- Le Daily YAP est le réservoir qui alimente la cadence (IG 14 reels/sem, YT 1 long + 3-4 Shorts, LI 4 posts).
- Le Script Builder est le seul process qui transforme une idée parlée brute en script filmable — clé pour la production réelle de Chris.
