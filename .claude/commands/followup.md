# Séquence Follow-Up BA™

Génère les 3 messages de relance pour : `$ARGUMENTS`

## Workflow

1. Lis [02-SALES/script-closing-v2.md](02-SALES/script-closing-v2.md) — gestion des "pas maintenant"
2. Lis [02-SALES/objections-repertoire.md](02-SALES/objections-repertoire.md) — objections et réponses
3. Lis [02-SALES/dm-sequence-linkedin-instagram.md](02-SALES/dm-sequence-linkedin-instagram.md) — patterns de relance

## Règles ABSOLUES

- JAMAIS de pression ou countdown factice
- JAMAIS "t'as pris ta décision ?"
- Partir de quelque chose de concret (ce qu'il a dit, son business, un post récent)
- Tutoiement · direct · respect · généreux
- Message 3 = fermer la boucle proprement, oui ou non

## Format de sortie

```
MESSAGE 1 (J+2) — Check-in naturel :
[1-2 phrases, question ouverte, zéro pression]

MESSAGE 2 (J+7) — Valeur ajoutée :
[Observation sur son business ou ressource utile + 1 question]

MESSAGE 3 (J+14) — Dernière fenêtre :
[Transparence totale, fermer proprement, respecter sa décision]
```

Ton : direct · chaleureux · sans manipulation.
