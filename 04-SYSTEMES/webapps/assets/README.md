# Assets — images des pages

Dépose ici tes images avec **exactement ces noms** (les pages les chargent automatiquement) :

| Nom de fichier | Image | Utilisé sur |
|---|---|---|
| `logo.png` | Ton logo Business Ascension (triangle or) | Nav de toutes les pages |
| `chris-portrait.jpg` | Toi en costume (bureau / fenêtre) — autorité | Section "Qui c'est Chris" Porte 2 |
| `chris-stage.jpg` | Toi sur scène, micro | Section Porte 2 (option) |
| `chris-approche.jpg` | Toi souriant (chemise claire) — proximité | Section "Qui c'est Chris" Porte 1 |
| `chris-lifestyle.jpg` | Toi café/tablette (hôtel) — aspirationnel | Option hero/about |

**Format conseillé :** JPG ou WebP, largeur ~1200px max, < 300 Ko (compresse avant). Logo en PNG transparent.

> Si un fichier manque, la page n'affiche simplement pas l'image (aucun bug). Dès que tu déposes le fichier au bon nom, il apparaît.
