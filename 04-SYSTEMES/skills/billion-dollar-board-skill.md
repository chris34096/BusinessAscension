## EXPERT 1: ALEX HORMOZI - THE OFFER ARCHITECT

### Core Philosophy
"Make your offer so good, people feel stupid saying no."

### Key Frameworks

#### THE VALUE EQUATION
Value = (Dream Outcome x Perceived Likelihood) / (Time Delay x Effort & Sacrifice)

Augmenter : Dream Outcome + Perceived Likelihood
Diminuer : Time Delay + Effort

#### THE GRAND SLAM OFFER
1. Lister tous les problemes de l'ICP
2. Creer une solution pour chacun
3. Bundle : Core + Garantie + Scarcite + Bonuses + Nom de marque

#### THE PRICING FRAMEWORK
- Entry: 500-2K | Core: 5K-25K | Premium: 25K-100K+ | Elite: 100K+

**BA mapping :** Starter (997-2497) = Entry / IAP beta (4997) = Core / IAP standard (7K-12K) = Core-Premium

#### LTV MAXIMIZATION
Ascension -> Frequency -> Retention -> Referrals

**BA sequence :** Starter -> IAP Signature -> IAP Blackroom -> Scale Ascension

## EXPERT 5: JORDAN BELFORT - THE SALES PSYCHOLOGY EXPERT

### Core Philosophy
"Every sale is the same: a transfer of certainty from you to your prospect."

### Key Frameworks

#### THE STRAIGHT LINE SYSTEM
Prospect incertain -> Prospect certain (sur 3 axes)

**The Three Tens :**
1. **Product Certainty** : "Ca va resoudre mon probleme"
2. **Trust Certainty** : "Je fais confiance a cette personne"
3. **Company Certainty** : "Cette entreprise va delivrer"

Si un score < 8/10 -> impossible de closer. Identifier lequel est bas.

#### THE LOOPING TECHNIQUE
1. Objection surface
2. Acknowledge + identifier la vraie certitude basse
3. Loop back (story/case study) pour remonter cette certitude
4. Re-tenter le close
5. Repeter 3-5 fois

**BA application :** Utiliser dans script-closing.md - chaque objection = loop vers un cas client ou la garantie

#### THE TONALITY FRAMEWORK
- **Absolute Certainty** (produit/features) : profond, confiant, zero hesitation
- **Reasonable Man** (prix/logistique) : conversationnel, comprenant
- **Empathy/Sincerity** (pain points) : lent, doux, genuinement bienveillant
- **Mystery/Intrigue** (value propositions) : curiosite, secretive

#### OBJECTION TYPES
- Smoke Screens -> creuser la vraie objection
- Stalls -> urgence + loop
- Real Objections -> contrebalancer avec la certitude
- Fatal Flaws -> disqualifier avec grace

## EXPERT 7: DAN KENNEDY - THE HIGH-TICKET MARKETING SPECIALIST

### Core Philosophy
"Make money, not excuses."

### Key Frameworks

#### MESSAGE-MARKET-MEDIA MATCH
Les 3 doivent etre alignes. Diagnostiquer lequel est casse :
- Wrong Message -> bon marche, bonne plateforme, message qui ne resonne pas
- Wrong Market -> excellent message, mauvaises personnes ciblees
- Wrong Media -> bon message, bonne cible, mauvaise plateforme

**BA diagnostic :** Message (dents de scie -> simultaneite) FORT. Market (entrepreneurs FR avec revenus) PRECISE. Media (LinkedIn + Instagram) A VALIDER par les KPIs.

#### MAGNETIC MARKETING SYSTEM
1. **Capture Attention** (lead magnet irrésistible)
2. **Educate & Build Authority** (sequence multi-etapes)
3. **Make Irresistible Offer** (avec risk reversal)
4. **Follow Up Relentlessly** (7-12 touches avant achat)

#### HIGH-TICKET POSITIONING - 5 PILLARS
1. Exclusivite (pas pour tout le monde - criteres candidature IAP)
2. Autorite (prescription, pas suggestion)
3. Focus resultats (outcomes + garantie 30j)
4. Presentation premium (chaque touchpoint poli)
5. Pricing premium (jamais s'excuser de 4997)

#### KENNEDY SALES LETTER FORMULA
Headline -> Lead/Opening -> Story -> Offer -> Proof -> Stack -> Price Reveal -> Guarantee -> Scarcity -> CTA -> P.S.

**BA application :** Structure utilisable pour page-principale-copy.md (deja construit en 7 sections)

## EXPERT 8: TONY ROBBINS - THE PEAK PERFORMANCE STRATEGIST

### Core Philosophy
"The only thing that's keeping you from getting what you want is the story you keep telling yourself."

### Key Frameworks

#### THE SIX HUMAN NEEDS
1. **Certainty** -> garanties, processus prouves, cas clients
2. **Uncertainty** -> innovation, nouveaux outils, breakthrough
3. **Significance** -> acces exclusif, "un parmi 5 clients ce trimestre"
4. **Connection** -> communaute Starter, relation directe Chris
5. **Growth** -> transformation, deblocage, evolution identitaire
6. **Contribution** -> impacter d'autres, laisser un heritage

**BA ICP values primarily :** Certainty (systeme prouve) + Growth (CA previsible) + Significance (categorie a part)

**Contenu a creer :** Adresser ces 3 besoins dans chaque piece de contenu

#### STATE MANAGEMENT - THE TRIAD
1. **Physiology** (body) -> posture, respiration, mouvement
2. **Focus** (mind) -> ce sur quoi tu te concentres = ce que tu ressens
3. **Language** (words) -> langage autonomisant vs desautonomisant

**BA Inner Work mapping :** Breathwork = physiology | Hypnose/PNL = focus + language | MAP = recablage profond

#### RAVING FAN FORMULA
- Meet Expectations -> satisfait (pas de referrals)
- Exceed Expectations -> content (peut-etre recommande)
- **Create Raving Fans** -> evangeliste (refere, poste, reste)

**BA Raving Fan Protocol :**
- S1 : Onboarding personnel (pas automatise)
- S4 : Check-in appel (meme si tout va bien)
- S8 : Bonus surprise (ressource non attendue)
- Resultat booste : celebrer avec geste personnalise
- Ongoing : VIP treatment a chaque touchpoint

#### THE ULTIMATE SUCCESS FORMULA
1. Know Your Outcome (clair, mesurable, date)
2. Take Massive Action (biais vers la vitesse)
3. Know What's Working (mesurer tout)
4. Change Your Approach (pivoter jusqu'a l'outcome)

**BA application :** Objectif S1-S12 par client documente dans business-playbook-template.md
