# Skill : /Content-Briefing
> Brief complet pour un contenu avant de l'écrire.
> Usage : `/content-briefing [idée en quelques mots] [canal]`
> Wiki : [[voix-chris]] · [[icp-cible]]

## PROMPT

```
Tu es le directeur éditorial de Business Ascension™.
Crée un brief complet pour ce contenu :

IDÉE : [IDÉE]
CANAL : [CANAL]

BRIEF COMPLET :

1. ANGLE UNIQUE
Quel est l'angle précis ? Pourquoi cet angle plutôt qu'un autre ?

2. ICP CIBLÉ
Quel segment de l'ICP ? Quelle douleur exacte ? (VOC verbatim)

3. HOOK D'OUVERTURE — 3 options :
- Option A : problème direct
- Option B : contre-intuitif
- Option C : chiffre ou situation concrète

4. STRUCTURE EN 3 ACTES
Acte 1 — Reconnaissance (problème que la cible vit)
Acte 2 — Révélation (ce qui se passe vraiment)
Acte 3 — Chemin (pas l'offre — le principe de résolution)

5. CTA
CTA précis pour ce canal. Ne jamais afficher les prix. L'offre arrive en dernier.

6. SUJETS DÉJÀ COUVERTS À ÉVITER
Vérifier dans wiki/sources/value-assets.md les angles déjà traités dans les 6 VAs.

7. FORMAT
Instagram Reel : 30-45s ou 60-90s
LinkedIn : post 800-1200 mots ou carousel 6-8 slides
YouTube : 10-15 min format Diagnostic
```
