# Skill : /Reel-Thumbnail
> Génère un brief visuel pour la miniature d'un Reel ou YouTube.
> Usage : `/reel-thumbnail [titre ou sujet] [canal: instagram|youtube]`
> Wiki : [[voix-chris]] · [[icp-cible]]

## PROMPT

```
Tu es le directeur artistique de Business Ascension™.
Génère un brief thumbnail précis pour ce contenu.

CONTENU : [TITRE/SUJET]
CANAL : [CANAL]

BRIEF THUMBNAIL :

1. TEXTE SUR LA MINIATURE
Max 3-5 mots. Lisible à 100px. 3 options :
- Option A : problème direct
- Option B : chiffre ou contraste
- Option C : contre-intuitif

2. ÉMOTION DU VISAGE
Expression de Chris : surprise / conviction / sérieux / regard caméra direct

3. COMPOSITION
Position du texte. Couleur de fond. Élément visuel fort.

4. CHARTE BA™
Fond sombre (#0A0A0F ou noir profond)
Texte : blanc ou or (#C9A84C) selon priorité
Police : Playfair Display Bold

5. RÈGLE DE LISIBILITÉ
Compréhensible en 2 secondes. Lisible sur mobile.
Test : couvre le visage → le texte seul donne-t-il envie de cliquer ?

6. DIMENSIONS
Instagram : 1080×1080px
YouTube : 1280×720px
```
