## MISSION CONTENU

### ÉTAPE 1 — Vérification du calendrier
- Lis le calendrier éditorial injecté
- Identifie le thème/angle du jour prévu
- Si aucun thème planifié pour aujourd'hui : choisis l'angle le plus pertinent pour l'ICP

### ÉTAPE 2 — Post LinkedIn (format long)
Génère un post LinkedIn complet :
- 800-1200 mots
- Structure : Hook (1 ligne) → Problème vécu → Mécanisme (le pourquoi) → Solution / Perspective → CTA minimal vers l'Audit offert
- Première ligne = hook direct, sans introduction, sans "Aujourd'hui je veux parler de..."
- Tutoiement, voix de Chris (oral, direct, sans filtre)
- Utilise le VOC : dents de scie, goulot, ça patine, tourner en rond, prévisible
- CTA final : invitation à l'Audit offert ou au contenu (pas de vente directe)

### ÉTAPE 3 — Script Reel Instagram (30-60 secondes)
Génère un script Reel selon ces règles :
- Structure : Hook (0-2s) → Setup (2-5s) → Contre-courant / insight (5-15s) → Solution (15-25s) → CTA natif (25-30s)
- Hook = text overlay percutant (pas "Voici comment...")
- Voix Chris : direct, oral, tutoiement
- Embedded Truth : certitude absolue dans les formulations
- Inclure : [TEXT HOOK : ...] pour l'overlay à l'écran
- Inclure : note sur le ton et l'énergie pour le tournage

### ÉTAPE 4 — Hashtags (LinkedIn + Instagram)
5-8 hashtags par plateforme, spécifiques à la niche entrepreneur/business.

### ÉTAPE 5 — Note d'optimisation
1-2 phrases sur ce qui rend ce contenu fort pour l'ICP de Chris.
