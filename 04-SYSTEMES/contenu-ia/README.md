# Système de Contenu IA — Business Ascension™
> Inspiré du système Hormozi : 3 dossiers → contenu indistinguable du vrai Chris.
> Principe : l'IA sonne comme internet parce qu'elle a été entraînée sur internet. Entraîne-la sur TON travail → elle sonne comme TOI.

---

## STRUCTURE

```
contenu-ia/
├── business-context.txt       ← Colle ça EN PREMIER dans chaque session Claude
├── data/
│   ├── README.md              ← Comment alimenter les data (lis ça)
│   ├── linkedin-posts.txt     ← Tes meilleurs posts LinkedIn (à alimenter)
│   ├── instagram-captions.txt ← Tes meilleures captions Reels (à alimenter)
│   ├── youtube-scripts.txt    ← Tes transcripts YouTube validés (à alimenter)
│   └── voc-verbatim.txt       ← Langage exact de ta cible
└── prompts/
    ├── prompt-01-linkedin-post.md   ← Posts LinkedIn texte
    ├── prompt-02-reel-script.md     ← Scripts Reels 30-90s
    ├── prompt-03-youtube-script.md  ← Scripts YouTube 10-15 min
    └── prompt-04-carousel.md        ← Carousels LinkedIn 6-8 slides
```

## MODE D'EMPLOI — 3 ÉTAPES

### Étape 1 — Crée un Projet Claude permanent

Va sur claude.ai → Projets → Nouveau → "BA™ Contenu"

Dans les **fichiers du projet**, ajoute une fois pour toutes :
- `business-context.txt`
- `voc-verbatim.txt`
- Tes fichiers data (linkedin-posts, instagram-captions, youtube-scripts)

Ces fichiers deviennent le contexte permanent — tu n'as plus besoin de les recoller.

### Étape 2 — Colle le prompt adapté au format voulu

| Format | Prompt |
|---|---|
| Post LinkedIn | `prompt-01-linkedin-post.md` |
| Script Reel | `prompt-02-reel-script.md` |
| Script YouTube | `prompt-03-youtube-script.md` |
| Carousel LinkedIn | `prompt-04-carousel.md` |

Claude va : analyser tes données → extraire les patterns de ta voix → construire des templates → attendre ton idée.

### Étape 3 — Donne l'idée, reçois le contenu

**TOI tu fournis toujours l'idée. L'IA fournit l'exécution.**

> Hormozi : "L'idée est l'alpha. C'est là que sont les plus hauts rendements. L'IA peut la développer, mais l'idée de base est toujours la tienne."

Exemples d'idées à donner :
- "Idée : pourquoi les entrepreneurs qui postent tous les jours n'ont pas de clients"
- "Idée : la différence entre savoir quoi faire et le faire de façon constante"
- "Idée : ce que j'ai compris en accompagnant 350 personnes sur le blocage intérieur"

Reçois 5-10 drafts. Dis à Claude lesquels tu aimes / tu n'aimes pas → il apprend et s'améliore.

---

## RÈGLE ABSOLUE

**Ne jamais demander "fais-moi un post LinkedIn"** sans :
1. Le business context (dans le projet)
2. Les data (tes contenus passés)
3. Le prompt spécifique (long, précis)
4. Une idée concrète de ta part

Sans ces 4 éléments → AI slop garanti.

---

## ALIMENTER LES DATA AU FIL DU TEMPS

Chaque contenu publié qui performe → l'ajouter dans le fichier data correspondant.

- Post LinkedIn qui cartonne → copie dans `linkedin-posts.txt` avec nb de likes/commentaires
- Caption Reel qui génère des DMs → copie dans `instagram-captions.txt`
- YouTube publié → exporte le transcript depuis YouTube Studio → colle dans `youtube-scripts.txt`

Plus la bibliothèque grandit, plus le résultat est précis.
