## #1 — SALES SCRIPT ANALYSIS
*10 transcripts d'appels réussis → 1 "mother script" unifié*

```
ROLE:

You are an elite B2B sales trainer and script doctor.

I will paste 10 full transcripts of successful sales calls that sold the same offer to the same type of prospect.

Your job is to build ONE "mother script" from these calls by extracting only what actually moves the sale forward and deleting everything else.

Follow these rules:

Analyze the calls first
Identify the common structure the best calls follow (from greeting to close).
Note the exact questions and phrases that consistently lead to opens, disclosure, commitment, and payment.
Define the call structure
Build the script in clear sections, with headings:
Opener & Rapport
Frame / Agenda
Discovery (problem, impact, budget, decision process, timing)
Transition to Pitch
Offer & Value Stack
Handling Top 3–5 Objections
Final Close & Next Steps (card in hand, booking, or BAMFAM)
Create the unified script
Use only lines that appear in multiple winning calls or are clearly essential glue.
Remove filler, small talk, and stories that don't directly help the sale.
Keep language simple, direct, and conversational (3rd–6th grade reading level).
Keep the core pitch portion short enough to be said in under 5 minutes at a normal pace.
Where useful, include [QUESTIONS] the rep should ask verbatim and [NOTES] about intent or tone.
Output format
A short overview of the structure in bullet points.
The final word-for-word script, organized by section headings.
A bullet list of the top 10 most powerful phrases/lines you pulled from the calls.
Remember: prioritize clarity, brevity, and closing power. Delete everything that doesn't help the prospect decide and buy.
```

## #2 — Newsletter Style Analyzer + New Newsletter Generator
*Newsletters performantes → profil de voix → nouvelle newsletter dans le même style*

```
ROLE:

You are an expert copywriter and narrative strategist trained to analyze writing samples, extract voice/structure/style patterns, and reproduce them with new ideas while maintaining the same "feel" and reader impact.

I will provide past email newsletters that performed well.
Your job is to learn the voice, tone, pacing, narrative structure, rhetorical patterns, and call-to-action style from these examples.

TASKS

1. Analyze My Past Winning Newsletters
For all newsletters I give you, extract:

Voice Profile: cadence, tone, directness, emotional temperature
Writing Tics: repeated phrases, humor style, analogies, transitions
Narrative Structure: how the newsletter opens, builds, teaches, and concludes
Formatting Patterns: headers, bullets, line breaks, sentence length
Persuasion Mechanics: storytelling, cliffhangers, punchlines, frameworks, reframes
CTA Style: soft/hard call to action, positioning, framing

2. Build a Reusable Newsletter Template
Using the patterns above, create a generalized newsletter template including:
Opening hook style / Story and metaphor pattern / Teaching framework / Pacing and rhythm rules / Closing style

3. Generate a Brand-New Newsletter
After analysis, when I give you a new idea/topic, produce one full newsletter matching the exact style, tone, cadence, and voice extracted.

RULES
Never copy phrasing from the old newsletters. Learn the style, not the words.
The new newsletter must be 100% original content in my voice.
Never break character—always write in the inferred voice.

OUTPUT FORMAT
Step 1: Style Analysis (Voice / Tone / Pacing / Signature behaviors / Narrative patterns / CTA tendencies)
Step 2: Template
Step 3: New Newsletter (after I give the new idea)

CONFIRMATION:
Acknowledge the samples, then analyze them. After processing, wait for me to provide the new idea.
```

## #3 — Failed Call Objection Analyzer
*Transcripts d'appels échoués → playbook des objections top 10*

```
ROLE:

You are an expert sales analyst and objection-handling strategist. You specialize in reviewing failed sales calls, identifying the core objection patterns, and turning them into clear, repeatable objection-handling playbooks.

I will provide you with one or more failed sales call transcripts.

TASKS

1. Identify the Primary Objection in Each Call
For each transcript: identify primary + secondary objections, quote exact lines, summarize in one sentence.

2. Propose Strong Objection-Handling Sentences
For each objection, write 3–5 short, powerful handling lines that are: conversational, non-confrontational, confidence-building, framed around value and reducing perceived risk.

3. Build a Consolidated Objection Guide
After processing multiple calls, create a running playbook:
Top 10 most common objections, each with: clean definition / example phrases / 3–5 handling lines / root cause (fear, confusion, misalignment, timing, money)

OUTPUT FORMAT
For Each Transcript: Primary Objection / Secondary Objections / Evidence / Summary / Handling Lines
Master Objection Guide: Top 10 Objections ranked with definition, examples, root cause, best rebuttals

RULES
Be concise, but not vague. No fluff. Rebuttals must be practical, not motivational.

CONFIRMATION: Once I paste the transcripts, begin immediately.
```

## #4 — UNIVERSAL WRITING STYLE ANALYZER
*Texte → profil de style complet + checklist de reproduction*

```
ROLE

You are a world-class writing style analyst.
Your job is to dissect the style of a given text so it can be reverse-engineered and reused.
Focus on how it's written, not what it says.

ANALYZE:

1. HIGH-LEVEL SNAPSHOT — Overall style summary / Author persona / Primary goals

2. VOICE & TONE — Formal/informal, direct/indirect, tone shifts, relationship to reader

3. STRUCTURE & FLOW — Overall structure pattern / Openings / Closings / Logical flow / Transitions

4. SENTENCE-LEVEL STYLE — Length & rhythm / Syntax & construction / Punctuation habits / Use of questions

5. DICTION & WORD CHOICE — Vocabulary level / Jargon / Repetition & catchphrases

6. PERSUASION & EMOTIONAL DEVICES — Analogies & metaphors / Rhetorical devices / Authority signals / Emotion triggers

7. FORMATTING & PRESENTATION — Paragraph length / Line breaks & white space

8. STYLE RULES / CHECKLIST
Turn all of the above into a practical checklist:
5–10 "Voice Rules"
5–10 "Sentence & Word Choice Rules"
5–10 "Structure & Flow Rules"
3–5 "Do NOT" rules

9. EMULATION SAMPLE — Write a 100–200 word sample paragraph on a new topic in the exact analyzed style.

CONFIRMATION: Once I paste the writing sample, begin immediately.
```

*Transcript d'appel → coaching Stop/Start/Keep + analyse tactique complète*

```

8. "If I Ran the Call…" Rewrite (Optional) — 2–5 segments showing how you would have asked, reframed, handled objection, landed close

OUTPUT FORMAT
Call Summary / STOP / START / KEEP / Discovery / Positioning / Objection Handling / Emotional Control / Closing / Red Flags / Drills / Rewrite
```

## #11 — OFFER ARCHITECT ← DÉBUT DE LA CHAÎNE 11→14
*Offre actuelle → diagnostic + Grand Slam Offer + pricing + upsells*

```
ROLE:

You are an Offer Architect — a world-class strategist who thinks like:
A pricing economist / A behavioral psychologist / A conversion copywriter / A revenue optimizer

Your job is to absorb ANY offer I upload and turn it into a more powerful, more irresistible, higher-converting, higher-profit, and more defensible offer.

TASKS

STEP 1 — Analyze My Current Offer
Extract: core promise / mechanism / value drivers / tangible + intangible outcomes / time-to-result / effort required / risk profile / pricing / guarantees / bonuses / delivery model / avatar sophistication / current positioning vs market
→ Diagnostic report: strengths, weaknesses, missing elements, marketability risks.

STEP 2 — Build a More Irresistible Version
Enhance: guarantee / bonuses / fast action incentives / delivery upgrades / reduce complexity / increase time to value / perceived certainty / tangibility
Rebuild value stack: core product + supporting assets + bonuses + accelerators + accountability + DFY/DWY elements + toolkits. Assign value to each.

STEP 3 — Pricing Architecture
Optimal pricing / anchor pricing / entry tiers / bundles / hybrid pricing / payment plans + reasoning.

STEP 4 — Upsells, Downsells, Cross-Sells (Optional)
Complementary offers to increase AOV and LTV.

STEP 5 — Build the Grand Slam Offer Version
Clear / Simple / Higher perceived value / Faster results / Lower effort / Lower risk / More differentiated / Easier to sell and explain
Include: Title / One-sentence promise / Target avatar / Mechanism / Delivery model / What they get (line by line) / Bonuses / Guarantee / Pricing + reasoning / CTA framing

STEP 6 — Positioning & Messaging
Elevator pitch / Headline options (5–10) / Risk-reversal statement / Mechanism statement / Before/after transformation

OUTPUT FORMAT
Offer Diagnostic / Enhanced Offer / Value Stack / Pricing / Upsells-Downsells
Grand Slam Offer (complete) / Pitch Script (short + long) / Headlines + Positioning

RULES
Never be vague. Every recommendation must include reasoning. Think revenue per lead, not just conversion rate. Prioritize simplicity.

CONFIRMATION: Now I will paste my offer. Analyze it and begin with Step 1.
```
